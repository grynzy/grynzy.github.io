How to make cortical and cancellous bone models - PART 1: Meshmixer

The bone model is assumed to have cortical layer with certain thickness, and cancellous layer beneath it.

Introduction

Nowadays, we can simulate the stability test of bone implants without doing the real experiment by using simulation software. If in the real experiment we use cadaveric specimens or synthetic bone as well as implant models, then in the simulated environment we use virtual 3D model of the bone or the implant. In this post, I would like to share you my experience in preparing the bone model which also quite similar with other paper's method.

I used Meshmixer to perform this tutorial, which you can download it for free from this link

In my case, I used tibia bone and planned to make cortical layer (outer layer of bone) with a thickness of 3 mm across tibia surface, mimicking a paper in 2021 published in Nature.

Tibia bone model

Here I used tibia which I got from 3D scanning the synthetic tibia model. If you are also interested to use tibia or other bone models, there are several websites that give freely downloadable file of 3D bone model, such as this link.

Previously I already cut the tibia into half using the plane cut feature in Meshmixer as well, leaving the remaining tibia (from tibia plateau to middle part of the tibia).

The following method will comprise mainly these parts:

  • Offsetting
  • Separation of two layers
  • Side story - inspection

    • Offsetting

    Choose SELECT and use CTRL+A to select all region, the model color should be changed into different color by now (orange in my model).

    Selecting bone surface

    Then choose EDIT ‚Üí OFFSET, Meshmixer will solve it for a while, and after that, Meshmixer will display something like this:

    Offset feature parameters

    You can adjust the DISTANCE that represents how thick the cortical layer you want to have. By referring to some scientific articles, I set the thickness of my cortical bone model to be 3 mm.

    Be careful, this DISTANCE means the new layer will be created x mm away from the orange layer (went bigger by x mm than the original layer, in the case of picture above). If you want to have the new layer to be x mm smaller than the original one, you can choose either these ways:

    1a. Minus-ing

    Set the DISTANCE to - x mm. In my case, I set my bone model to be -3 mm, then press ENTER.

    Before (top) and after (bottom) the OFFSET using negative value

    1b. Flip Normal-ing

    You can swap the orange and the ‚Äúzebra‚Äù layer by choosing FLIP NORMAL first before choosing OFFSET. After flipping the sides, you can go to OFFSET and you can set the distance using the positive value (i.e. just as simple as 3 mm).

    Making of offset by Flip Normal function

    As reminder, the result using FLIP NORMAL will also have flipped side of orange and ‚Äúzebra‚Äù layer, which later you can re-flip it again for other purpose you would like. The result is as shown below.

    Bone model after re-flipping it

    Side Story - Failures

    If you want to try enclosing the two layers as well, you can follow these steps:

    Try to separate the two layers. Firstly click EDIT and hit SEPARATE SHELLS

    Separate Shells function

    The result was shown below:

    Two de-fused layers

    You can identify which part you are working now by identifying the lighter grey and its corresponding object browser.

    Then, for each shell, you can select ANALYSIS ‚Üí INSPECTOR. The blue ball indicate the surface we would like to close (repair) and use ‚ÄúFlat Fill‚Äù as its Hole Fill Mode. In here, I set the ‚ÄúSmall Thresh‚Äù to be 0 mm, to get the flattest result.

    Once you finish adjusting the small thresh and choosing the ‚ÄúHole Fill Mode‚Äù, hit "Auto Repair All"

    Cancellous bone from Inspetor method

    The same method can apply to the outer layer:

    Cortical bone from Inspector method

    As we can see, even though by using the 0 mm small thresh, the interface between the cortical and cancellous cannot be distinguished too clearly. Even the pink part of cortical bone "ate up" some portion of cancellous bone, referring the the surface has not been flatten well.

    Another way to make the thickened cortical layer is by ticking the ‚ÄúConnected‚Äù option during the OFFSET parameter adjustment. The result was like this:

    Cortical bone fixed with Connected method

    However, still, the result was not fully flat.
