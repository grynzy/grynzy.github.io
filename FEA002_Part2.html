How to make cortical and cancellous bone models - PART 2: Blender

The bone model is assumed to have cortical layer with certain thickness, and cancellous layer beneath it.

Introduction

This is the second post about my working flow (or I can say my experience) in making the cortical and cancellous bones. The first part of tutorial was about preparing the two layers of bone, using software called Meshmixer. If you are interested in the way I prepared the layers, you can check my previous post.

Rather than using Meshmixer again, I chose to use Blender software to finish up my model. You can download it for free from this link.

I put my arguments on fully finishing my work in Meshmixer on the last part of the first part of tutorial, too. Without further a do, here are the steps I did:

Set up the models and functions

Once you open Blender, just create new general model, and delete the cube that usually appears by default whenever you start new model. Then, import the bone model file by clicking FILE ‚Üí IMPORT ‚Üí any type of imported file. In my case, I used STL file which was exported from my first part of tutorial.

Imported bone model got from the first tutorial

In making the final result, I used principle of Boolean operation, where some parts are used as cutter for other parts to yield new modified part. Blender provides this function, but it may be easier to be used by open EDIT ‚Üí PREFERENCES. Then, go to ADD-ONS tab, and type ‚ÄúBool‚Äù in the search tab. The search result would look like this:

Enabling the Bool Tool

Once you enable it, the layout on the right side of the software would be like this:

Bool Tool Enabled

As I mentioned about Boolean, there must be a part to be the ‚Äúcutter‚Äù for other part. In my case, I used simple plane to be the cutter. I made the plane model while my layout is still in Object Mode, not Edit Mode. I hit ADD ‚Üí MESH ‚Üí PLANE. There will be a plane appearing on the center of coordinate.

I re-scaled the size of the plane as well as its position to the position where it intersected small part of the bottom part of my bone model (distal part of tibia). Indeed, the end result yield shorter bone model as the compensation of using this Boolean. You can re-scale the plane by hitting S after you click the plane. You can move the plane by the same manner but use G instead of S. In my case, after I hit G, I used Z so that the plane moved along the Z-axis.

Scaling the plane

Typing G and Z to adjust the plate position

Plane adjusted

I also separated the outer and inner bones by choosing the bone first. Then I hit TAB to change the mode into Edit Mode. With the bone being covered by orange dots (yeah there were so many dots or so-called nodes) I used left click ‚Üí SEPARATE ‚Üí BY LOOSE PARTS.

Separating the bone layers

It ended up giving me two separated bones.

As the Boolean operation would delete the original file, I suggested to have copies of each model and keeping them in new collection as ‚ÄúBackup files‚Äù.

Now that the file has been imported and the tools needed has been enabled, what remained was the Boolean operation

Boolean Operation

The principle in using the BOOL TOOL is to click one part, hold SHIFT while clicking other part, and perform one of the AUTO BOOLEAN. In my case, firstly I only made the models I want to involve in as the visible ones, leaving other models being hidden. I clicked the plane first, then held SHIFT while choosing the outer bone model. You can now see that the first one being chosen had darker color.

Boolean operaton for outer bone

I opened EDIT tab at the right side of layout, expanded BOOL TOOL, and chose SLICE. The result was shown below:

Result of Boolean operation

It was not finished yet. The orange surface cultivated from the Boolean operation then was deleted, giving final result like this:

Obtained outer bone model

I would like to remind you that the plane was deleted after this process. That is where backing up other plane took its role. Same procedures were also applied to the inner bone model.

Boolean operation for inner bone

Result of Boolean operation on inner bone model

Finished Boolean operation of inner bone model

As we can see, there were some flakes of grey shade on the inner bone model because both first and second results surface collided each other. It brought me to the last step, where I firstly chose inner bone, held SHIFT while choosing the outer one, and hit the SLICE on Auto Boolean.

Last Boolean operation

The final result was eventually obtained.

Final result
